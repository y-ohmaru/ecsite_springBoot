ECサイト
概要
Spring Boot を使用して作成した ECサイトのサンプルアプリケーションです。
ユーザー認証、商品一覧、カート機能、注文処理など、ECサイトの基本機能を実装しています。

Spring Boot を用いた **MVCアーキテクチャおよび3層構造（Controller / Service / Repository）**で設計しています。

## 使用技術

### Backend
- Java 21
- Spring Boot 3.4.2
- Spring Web
- Spring Security
- Spring Data JDBC

### Frontend
- Thymeleaf

### Database
- MySQL
- H2 Database（開発環境）

### Validation
- Bean Validation

### Object Mapping
- ModelMapper

### Testing
- JUnit
- Mockito

### Build Tool
- Maven

### その他
- Lombok
